Core Features

Feature 1: AI Letter Generation

User selects a style profile
User types instructions describing the letter they need
App sends request to backend API
Generated letter appears in preview area
Letter can be refined/regenerated
Feature 2: Style Profile Training

User uploads 3-6 sample letters (JPEG images or PDF)
Files are sent to backend for AI analysis
Backend extracts writing patterns, tone, and style
Profile is saved for future letter generation
Feature 3: PDF Generation with Custom Fonts

Generate professional A4 PDF documents
Embed custom Unicode fonts for proper character rendering
Support for English and Marathi (Devanagari script)
Automatic font selection based on content language
Include optional letterhead at top of PDF
Feature 4: Letterhead Management

Upload company/organization letterhead images
Save multiple letterheads with names
Select which letterhead to use for each letter
Letterhead appears at top of generated PDFs
Feature 5: Multi-language Support

English text rendering
Marathi (Devanagari) text rendering
Automatic language detection for font selection
Uses Noto Sans for English, Noto Sans Devanagari for Marathi
Backend API Integrations Required

API 1: Style Profile Analysis (n8n Webhook)

Endpoint: POST request
Sends: style_name, description, multiple file attachments
Purpose: Analyzes uploaded sample letters to learn writing style
API 2: Letter Generation (n8n Webhook)

Endpoint: POST request
Sends: profile_id, context (user instructions)
Returns: Generated letter text
Purpose: Creates new letter based on selected style and user instructions
API 3: Database (Supabase)

Table: letter_style_profiles (id, profile_name, description, created_at)
Table: letterheads (id, name, image_url, created_at)
Storage bucket: letterheads (for uploaded letterhead images)
